# Operations Management Dashboard

## Overview
This project showcases an **Operations Management Dashboard** built in **Power BI** to provide a unified view of inventory, workforce, sales, salary, and process KPIs.  
The report is designed for daily operations use, with both light and dark themes to support usability in different environments.

## Problem
The organization lacked a single, interactive view of operational performance across key areas such as inventory, workforce, sales, salary, and process KPIs.  
Without a centralized dashboard, it was difficult to quickly spot stock issues, high turnover, and delayed operations.

## Solution
An **Operations Dashboard in Power BI** that brings together operational metrics into one report, with a focus on:

- Data cleaning and preparation inside Power BI (Power Query)
- Data modeling and relationships
- KPI design for operations and workforce
- Clear page layout with light/dark theme variants

The dashboard provides a centralized operational view that is easy to read for managers and teams.

> Note: The data used is sample data for demonstration and training, not real company data.

## Report Pages
The report is structured into multiple pages, such as:

1. **Overview KPIs**  
   A high-level summary of inventory, workforce, sales, salary, and process indicators.

2. **Inventory and Stock View**  
   Visuals to monitor stock levels, negative stock, and potential inventory risks.

3. **Workforce Metrics**  
   Views related to headcount, turnover, and workforce distribution.

4. **Sales and Process KPIs**  
   Charts focusing on sales performance and process completion/on-time ratios.

## Tools and Techniques
- **Power BI Desktop**
- **Power Query** for data cleaning and transformation
- **Data modeling** (relationships, data types, hierarchies)
- **DAX** for calculated measures and KPIs
- **Theme design** for light and dark mode layouts

## Skills Demonstrated
- Data cleaning and preparation in Power BI
- Building an operational data model
- KPI and metric design for operations
- Dashboard layout and UX for daily monitoring
- Translating business requirements into visuals and measures

## How to Use
1. Open the `.pbix` file in **Power BI Desktop**.
2. Explore the different pages:
   - Overview
   - Inventory / Stock
   - Workforce
   - Sales & Process KPIs
3. Use slicers and filters to interact with the data and see how the dashboard responds.

## Notes
